# Software-Developer-Salary-Prediction
A Machine Learning Web-App to predict salary of a Software Developer in different countries.
[Click Here](https://devsalaryprediction.herokuapp.com/)

Libraries Used :
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn 
- Streamlit

Deployed on heroku
